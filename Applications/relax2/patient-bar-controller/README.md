Download and install Arduino IDE.



Copy the complete folder to your Computer (do not rename it).



Open the patient-bar-controller.ino file with Arduino IDE.



The library "StepperDriver" 1.5.0 should get installed automatically.

Check in the left library tab (filter by type: installed).

If not installed search for "StepperDriver" (1.5.0) in the library search bar on the left (filter by type: all).



Connect the Arduino Nano Every via USB.

Arduino Nano Every should be selected automatically (top left).

Select it again to select the port (COM...).

If it is connected Arduin Nano Every will change to bold font.



Compile the software and upload it to the Arduino Nano Every (top left (right) arrow).
